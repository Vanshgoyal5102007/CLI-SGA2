Justification
md5sum: I used this to calculate a unique hash for each file's contents, which is the most accurate way to detect duplicates regardless of what the files are named.
Redirection (2>>): This command takes any standard error messages and appends them to a separate log file, keeping the main terminal output clean.
Associative Arrays: I used a hash map (declare -A) to store the file hashes because it provides instant lookups to check if a file has already been processed.
